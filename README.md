Операционная система: Windows 10 x64
Оборудование:
Процессор: Intel(R) Core(TM) i7-4650U CPU @ 1.70GHz, 2301 МГц, ядер: 2, логических процессоров: 4
Видеокарта: Intel(R) HD Graphics 5000
Память: 8 Гб 

Управление:
клавиша		действие
1		Включается каркасный режим отображения полотна
2		Включается режим отображения нормалей полотна
3		Включается текстура для полотна и модель освещения Ламберта
4		Включается модель освещения Блинна (полотно текстурировано)
5		Включается прозрачность ткани (модель освещения Ламберта)

Ветер дует сам, меняя направление с определенной периодичностью.

Бонусы:
1) Реализована оригинальная форма ткани (в виде флага факультета)
2) Дополнено локальной моделью освещения Блинна (по кнопке 4)
3) Визуализация крепления (как меша, состоящего из треугольников)
4) Дополнительные простые меши в сцене (сфера на креплении и массив кубов - основание крепления)
5) Прозрачность ткани
